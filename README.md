# 𝐒𝐀𝐊𝐔𝐓𝐀 𝐌𝐃

WhatsApp Bot with Multi-Number Support & Telegram Pairing

## Features
- 🤖 WhatsApp Bot with multiple number support
- 🔗 Pairing via Web & Telegram
- 📱 Auto-reconnect on session expiry
- 🎨 Customizable themes
- 📊 Status & Statistics
- 🔔 Newsletter auto-follow
- 💬 Command handlers

## Deployment

### Environment Variables
- `BOT_NAME` - Bot display name
- `TELEGRAM_BOT_TOKEN` - Telegram Bot Token
- `TELEGRAM_OWNER_ID` - Your Telegram User ID
- `NEWSLETTER_JID` - Newsletter JID
- `OWNER_NUMBER` - Bot Owner WhatsApp Number

### Telegram Commands
- `/start` - Show menu
- `/pair <number>` - Get pairing code
- `/connect <number>` - Connect WhatsApp
- `/status` - Check bot status
- `/stats` - Show statistics (Owner only)
- `/restart` - Restart bot (Owner only)

### WhatsApp Commands
- `.menu` - Show main menu
- `.alive` - Check bot status
- `.owner` - Show owner info
- `.pair <number>` - Get pairing code
